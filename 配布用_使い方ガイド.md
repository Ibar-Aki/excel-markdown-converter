# Excel ↔ Markdown 変換ツール 使い方ガイド

このツールは、Excelの表とMarkdown（マークダウン）の表を、相互に変換する便利なツールです。
**ソフトのインストールは一切不要** で、WindowsパソコンとExcelが入っていれば誰でもすぐに使えます。

---

## 🔰 基本的な使い方（超簡単！）

`bat` フォルダの中にあるアイコン（バッチファイル）に、変換したいファイルを **ドラッグ＆ドロップ** するだけです！

### 1. Excelの表を Markdown にしたいとき

👉 **`excel2md.bat`** に Excelファイル（.xlsx など）をドラッグ＆ドロップしてください。
（同じフォルダに `.md` ファイルが自動的にできあがります）

### 2. Markdownの表を Excel にしたいとき

👉 **`md2excel.bat`** に Markdownファイル（.md）をドラッグ＆ドロップしてください。
（同じフォルダに `.xlsx` ファイルが自動的できあがります。罫線や色付きで綺麗に出力されます！）

---

## 🛠 その他の便利な変換ツール

`bat` フォルダには、用途に合わせた他の変換ツールも用意されています。
それぞれ、名前の通りの変換を行います。

- **`excel2csv.bat`**: Excel → CSV に変換（文字化けしないUTF-8形式）
- **`csv2excel.bat`**: CSV → Excel に変換（罫線・色付きの綺麗な表へ）
- **`csv2md.bat`**: CSV → Markdown に変換
- **`md2csv.bat`**: Markdown → CSV に変換

---

## ⚠️ 注意点（よくある質問）

- **Q. ツールを別の場所に移動したい**
  - A. `bat` フォルダと `scripts` フォルダは、**必ず同じフォルダ（セット）** に入れておいてください。片方だけ移動すると動きません。
- **Q. MarkdownからExcelに変換したとき、シートが複数できる**
  - A. Markdownの中に複数のテーブル（表）がある場合、自動的にExcelの別シートに分割して出力してくれます。
- **Q. エラーが出る / 変換が終わらない**
  - A. 変換元のExcelファイルを開きっぱなしにしているとエラーになることがあります。ファイルを閉じてから再度ドラッグ＆ドロップしてください。
